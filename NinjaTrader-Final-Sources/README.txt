These are the final licensed NinjaTrader source files for Good Gains products.

Use these files as the single source of truth for future product updates.

Files included here already contain:
- LicenseKey field support
- License validation against https://goodgainsindicators.com/api/verify-license
- Empty/invalid license blocking
- GGI license support

After editing any file:
1. Copy or import the updated .cs file into NinjaTrader.
2. Compile inside NinjaTrader.
3. Export a new ZIP via Tools -> Export -> NinjaScript Add-On.
4. Replace the matching ZIP in public/downloads.
5. Push the updated site/download changes to GitHub.
